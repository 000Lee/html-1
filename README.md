# VS code 단축키
* Home
행의 맨앞으로 이동

* End
행의 맨뒤로 이동

* Alt + ↑/↓
커서가 위치한 행 , 
선택한 행을 위 혹은 아래로 이동
(여러 행 대응 가능)

* Alt + Shift + ↑/↓
커서가 위치한 행, 
선택한 행을 복사하며 이동
(여러 행 대응가능)

* Ctrl + C, Ctrl + V, Ctrl + X
선택 범위를 복사 붙여넣기 잘라내기. 
블록 선택하지 않은 상태에서 사용가능.

* Ctrl + Enter
다음 라인으로 이동

* Ctrl + ], [
오른쪽 들여쓰기, 왼쪽 들여쓰기

* Ctrl + Shift + L
같은 글자 한꺼번에 선택, 수정 가능

* 주석
ctrl + /
코드에 대한 설명을 할 때 주로 쓰임
프로그램 영향을 미치지 X

* 탐색기  
(폴더 목록 접었다 펴기)
ctrl+b

* ctrl +` (backtick)
문제,출력,디버그콘솔,터미널,포트창 뜸

# 표기법 종류
* 카멜
주로 변수명, 함수명에 사용한다
userList, newTodoList

* 파스칼케이스
주로 클래스명, 객체명에 사용한다
UserList, NewTodoList

* 스네이크케이스
주로 파일명에 사용한다
user_list,new_todo_list

* 케밥 표기법
주로 css에 사용한다
user-list, new-todo-list

* 헝가리안 표기법
주로 jquery에 사용한다
&userList, $newTodoList
